A) HTML Structure - top

This is very simple for editing,Firstly you should add index.php your server .After this step if you want to change any elements in this plugin you should edit elements.json file. This file inludes all elements settings such as name,icon(font-awesome icons) and content url. You can see elements.json structure below 

{
  "elements": [{
      "name": "Media",
      "items": [{
          "name": "Image",
          "icon": "fa fa-picture-o",
          "content": "elements/Image-full.html"
      }
    ]
  }, {
      "name": "Footer",
      "items": [{
          "name": "Adress",
          "icon": "fa fa-minus",
          "content": "elements/address.html"
      }]
  }]
}

C) JavaScript - top

This theme imports four Javascript files.

  1. bal-email-editor
  2. colorpicker
  3. demo
  4. email-editor.bundle
  1. bal-email-editor is a main plugin for this file
  2. colorpicker is a colorpicker css for the main page
  3. demo for the load example template for editing
  4. email-editor.bundle contains all files need for the plugin

D) Php files - top

This plugin includes 8 php files.

  1. config is a settings for this plugin such as site directory,site url and etc.
  2. export for the save email builder result as zip(or html)
  3. get-files get all uploading files from uploads folder
  4. index main file this plugin, creating email builder page
  5. load_templates select all templates from database
  6. save_template for the save email builder result
  7. upload for the upload any image to uploads folder
  8. includes/db.class.php includes database connection and select ,insert queries

E) Sql files - top

dump.sql - all sql codes in here. Mainly you need table for email editor , second table - bal_email_limiter for set limit demo users .
